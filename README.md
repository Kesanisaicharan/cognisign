# 🤟 CogniSign

> **"When Every Second Matters, Every Sign Must Be Understood."**

CogniSign is an AI-powered real-time emergency communication platform that bridges the gap between deaf individuals and first responders — instantly, even without internet.

---

## 🚨 The Problem

During emergencies — accidents, medical crises, fires, floods — deaf and hard-of-hearing individuals face a critical communication barrier.

- ❌ Existing tools require typing
- ❌ Depend on internet connectivity
- ❌ Rely on interpreters
- ❌ Too slow for emergencies

**This communication gap can cost lives.**

---

## ✅ Our Solution

CogniSign uses a smartphone camera to:
- Recognize sign language gestures in real time
- Convert signs → text → speech for responders
- Convert responder speech → text → visual display for deaf users

**Complete two-way communication. Zero barriers.**

---

## ⚡ Key Features

| Feature | Description |
|--------|-------------|
| 🤟 Real-Time Sign-to-Speech | Signs instantly converted to spoken audio |
| 🎙️ Speech-to-Text | Responder speech shown as text on screen |
| 📶 Offline Edge AI | Works without internet in disaster zones |
| 🆘 Gesture SOS | Special gesture triggers alert + location share |
| 🏥 Smart Medical Profile | Blood group, allergies shared instantly with doctors |
| 🔴 Priority Detection | AI classifies Critical / High / Medium automatically |
